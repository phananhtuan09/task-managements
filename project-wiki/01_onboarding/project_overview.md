# Project Wiki System Overview

## Mission

The project wiki is a structured Markdown knowledge system for this repository. It acts as shared project memory so contributors can preserve durable context, align on intended behavior, and reduce repeated rediscovery.

## Who Uses It

- Human contributors who need onboarding, impact analysis, or durable reference material.
- AI agents that need stable context before drafting docs, updating docs, or changing code.

## Current Pilot Scope

The current MVP focuses on:

- root navigation and onboarding
- governance and update rules
- the first durable business-rules document for the wiki system
- folder entry points for future feature, flow, and decision docs

## What Later Phases Add

Later work will expand the folder model, add reusable templates, seed canonical content, introduce workflow commands and skills, and add verification support.
