# TaskFlow SaaS Wiki

## What This Wiki Covers

TaskFlow SaaS is a lightweight task management SaaS that helps teams organize work, assign tasks, and collaborate in real time.

## Start Here

- Read [INDEX.md](INDEX.md) for the wiki navigation.
- Read [Project Overview](01_onboarding/project_overview.md) for the product purpose and users.
- Read [Business Context](01_onboarding/business_context.md) for the problem statement and product value.

## Structure Overview

- `01_onboarding/` for product orientation and high-level context.
- `02_requirements/confirmed/` for durable product scope, business rules, and constraints.
- `03_features/` for canonical feature behavior and scope.
- Supporting wiki infrastructure exists, but it is not part of the normal product reading path.

## Scope Boundaries

This wiki is used as a lightweight product knowledge base for TaskFlow SaaS. Technical design details, API contracts, and implementation-specific docs should live outside this wiki.
